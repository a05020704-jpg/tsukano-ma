# 日記（つかのま）

1日15分だけ書く、静かな日記アプリ

## 特徴

- 📝 毎日15分だけ書く時間が決まっている
- 💾 ブラウザのローカルストレージに自動保存
- 📱 PWA対応 - ホーム画面にインストール可能
- 🔌 オフライン対応 - インターネット接続なしで動作
- 🎯 シンプルで落ち着いたUI

## インストール方法

### ウェブ版で使う

1. https://a05020704-jpg.github.io/tsukano-ma/ にアクセス
2. ブラウザのアドレスバーの「インストール」ボタンをクリック
3. アプリがホーム画面に追加されます

### 主な機能

- **今日**: 今日の日記を書く（最大15分）
- **振り返り**: 過去の日記を閲覧・編集
- **自動保存**: 書いている内容は自動で保存されます
- **オフライン対応**: インターネットがなくても使用可能

## 技術

- HTML5 + CSS3 + Vanilla JavaScript
- Service Worker（オフライン対応）
- Web Manifest（PWA対応）
- localStorage（データ保存）
